# Changelog

## 1.3 (versionCode 5)
### Added
- **Video playback** in the detail view (ExoPlayer / Media3)
  - Large centered play button on open — no autoplay
  - Tap to play / pause
  - Scrubber bar with current position and total duration
  - Seek thumbnail preview above the scrubber thumb while dragging
  - Mute / unmute button
  - Playback stops at end (no loop)
- Video grid cells keep the existing duration badge and play icon overlay

## 1.2.1 (versionCode 4)
### Fixed
- Crash when loading certain image formats during charging

## 1.2 (versionCode 3)
### Added
- Detail view: full-screen photo viewer with hero transition
- Pinch-to-zoom, double-tap to zoom, swipe-down to dismiss
- Bottom action bar: share, favorite, info, delete (stubs)
- Swipe between photos in the detail view (ViewPager2)

## 1.1 (versionCode 2)
### Added
- Grid polish: date section headers (sticky), year separators
- Photo and video counts in footer
- Favorite heart badge on grid cells

## 1.0 (versionCode 1)
### Added
- Initial release
- Photo & video grid (Library tab)
- Bottom navigation: Library, For You, Albums, Search
- Auto-update checker via GitHub releases
